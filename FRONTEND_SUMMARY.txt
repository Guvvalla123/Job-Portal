1. WHAT THE FRONTEND IS
CareerSync (branded in index.html title/meta and config/site.js default VITE_SITE_NAME)
is a React 19 SPA for a job portal: public job and company discovery, candidate
dashboard (profile, applications, saved jobs, job alerts, resume, data export),
recruiter workspace (jobs, companies, applicant pipeline, interviews), and admin
console (platform stats, users, jobs, companies, applications, audit logs, MFA security).

5. AUTH SYSTEM
Token storage:
  Access JWT: in-memory only (`accessTokenMemory` in apiClient.js), plus axios default Authorization header.
  Optional localStorage keys `currentUser` and `accessToken` (AUTH_KEYS) cleared on logout; user snapshot uses
  toPersistedSessionUser — not trusted for auth until getMe runs.
  Refresh token: httpOnly cookie `jid_refresh` (not readable by JS); rotation via POST /api/v1/auth/refresh.

Bootstrap flow (AuthContext.jsx):
  On mount: if SESSION_ENDED not set → restoreSessionFromCookie() → single-flight refresh → endBootstrap gate.
  Then GET /auth/csrf-token (best-effort), then getMe() → setUser or clear on failure.

Refresh flow:
  authRefreshCoordinator: one global POST /auth/refresh; 401 on API requests triggers getRefreshPromise(), retry once with _retry.
  peekInFlightRefresh lets requests wait for ongoing refresh without duplicate POSTs.

Logout flow:
  performLogout: POST /auth/logout with Bearer if present (skipGlobalErrorToast), setSessionEndedFlag, clearSession + clearPersistedAuthKeys,
  queryClient.clear(), window.location.replace('/login').

Session expiry:
  apiClient 401 handling clears session + toast; emitSessionExpired → AuthContext replaces to /login?session=expired.
  Cross-tab: storage listener on SESSION_ENDED_KEY.

6. API LAYER
apiClient.js:
  baseURL: VITE_API_URL resolved to `/api/v1` in dev or full URL with `/api/v1` suffix (see resolveApiBase).
  withCredentials: true; timeout 15000 (120000 for resume blob GET).
  Request interceptor: await bootstrap promise; await in-flight refresh; set X-CSRF-Token from document cookie;
    set Bearer from memory; strip Content-Type for FormData; Accept for blob GETs.
  Response interceptor: 401 → refresh + retry once; toasts for 403/5xx unless skipGlobalErrorToast; ECONNABORTED message rewrite.

authRefreshCoordinator.js — createAuthRefreshCoordinator(deps): getRefreshPromise, peekInFlightRefresh, resetForTests.

userApi.js — getMe, updateProfile, uploadProfileImage, uploadResume, deleteResume, getSavedJobs, toggleSavedJob, deleteAccount, changePassword, fetchMyDataExport, triggerDownloadDataExport.

jobsApi.js — listPublicJobs, getJobById, listMyJobs, getRecruiterAnalytics, getRecruiterApplicationTrend, createJob, updateJob, deleteJob.

applicationsApi.js — listMyApplications, applyToJob (Idempotency-Key header), listApplicationsForJob, getApplication, updateApplicationStatus, updateRecruiterNotes, updateInterview, listUpcomingInterviews, fetchApplicationResume.

companyApi.js — listMyCompanies, createCompany, updateCompany, deleteCompany, uploadCompanyLogo.

adminApi.js — getStats, getStatsTrend, getUsers, updateUserRole, toggleUserStatus, deleteUser, getJobs, updateJobStatus, toggleJobStatus, deleteJob, getCompanies, deleteCompany, getApplications, getAuditLogs.
