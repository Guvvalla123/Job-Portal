PROJECT OVERVIEW
---------------
A full-stack MERN (MongoDB, Express, React, Node.js) job portal application
with three user roles: Candidate, Recruiter, and Admin. Built for production
readiness with security, validation, and modern UX features.

Location: C:\Users\saiki\Desktop\Fullstack\Projects\job-portal-mern


TECH STACK
----------
Backend:  Node.js, Express 5, MongoDB (Mongoose), JWT, bcryptjs, Zod, Helmet,
          CORS, rate limiting, compression, nodemailer, Cloudinary, Multer
Frontend: React 19, Vite 7, React Router 7, TanStack Query, Tailwind CSS v4,
          React Hook Form, Zod, Sonner (toast), react-helmet-async


PROJECT STRUCTURE
-----------------

    constants/    roles.js
    controllers/  authController, userController, jobController, applicationController,

    services/     jobAlertService.js (matches jobs to alerts, sends emails when SMTP configured)
    utils/        jwt.js, asyncHandler.js, apiError.js, mail.js
    validations/  authValidation, jobValidation, companyValidation, applicationValidation
    app.js, server.js

  src/
    api/          apiClient.js (axios instance with interceptors)

    pages/        HomePage, LoginPage, RegisterPage, ForgotPasswordPage, ResetPasswordPage,
